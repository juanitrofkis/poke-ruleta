Poké-Ruleta Interactiva

Configura tres opciones, genera tu ruleta interactiva y obtén un Código QR dinámico. Aloja este archivo HTML en la web (por ejemplo en GitHub Pages o Netlify) para que tu amigo en Madrid pueda escanear el código y acceder a esta misma configuración al instante.

1. Configuración de Datos

En esta área de configuración, puedes ingresar los datos base que alimentarán la visualización. Al introducir los nombres, el sistema actualiza el estado de la aplicación, recarga el gráfico de la ruleta y regenera el enlace para compartir.

2. Compartir (Código QR)

Este código contiene la URL con tu configuración exacta. Pide a tu amigo que lo escanee.

3. Visualización Interactiva

Esta sección principal interactiva presenta la ruleta. Proporciona una herramienta visual dinámica que permite a los usuarios interactuar con los datos configurados, visualizar el sorteo y obtener un resultado claro basado en los parámetros de la URL.
